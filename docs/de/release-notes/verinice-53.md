# verinice 53

Die folgenden Neuerungen stehen Anwenderinnen und Anwendern mit dem Release von **verinice 53** zur Verfügung.

## CSV-Import 2 - Freitextfelder
<!-- epics/313 -->

Die zweite Iteration des **CSV-Imports** ermöglicht:
- den Import von Freitextfeldern.
- die Auswahl der Codierung.
- die Validierung der Daten vor Import.
- die Vorgabe eines Status.
