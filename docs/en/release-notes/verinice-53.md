# verinice 53

The following new features are available to users with the release of **verinice 53**.

## CSV Import 2 - Free-text Fields
<!-- epics/313 -->

The second iteration of the **CSV Import** enables:
- the import of free-text fields.
- the selection of encoding.
- validation of data before import.
- specifying a status.
