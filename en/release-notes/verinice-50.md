# verinice 50

The following new features are available to users with the release of verinice 50.

## Detail improvements and bug fixes

- Fixed a bug when creating risks that caused sporadic failures in the production environment due to unnecessary generation of domain IDs.
- In the ISO 27001 (DE) domain, only reports related to this domain are now displayed.
- The page layout has been improved in unit management and report view.
- A more meaningful icon is used for the units.
- The message when account creation fails has been made more specific.
- The positioning of messages in user management has been improved.
- Correction of some terms in the context help.
- In the documentation, all contrasts have been adjusted and some links have been correctly marked to improve accessibility.
- The documentation has been supplemented with a description of the versioning of domain templates and domain updates.
